# Creating Microservices for Account and Loan

## Objective

Create two independent Spring Boot microservices.

## Microservices

### Account Service

Endpoint

GET /account

Output

Account Microservice Running

### Loan Service

Endpoint

GET /loan

Output

Loan Microservice Running

## Technologies

- Java
- Spring Boot 3
- Maven

## Result

Successfully created Account and Loan microservices.
