# Exercise 1 - Create a Spring Web Project using Maven

## Objective
Create a Spring Boot web project using Maven.

## Technologies Used

- Java 23
- Spring Boot 3
- Maven

## Dependencies

- Spring Web

## Run

```bash
mvn spring-boot:run
```

## Expected Output

```
Tomcat started on port 8080
Started SpringRestHandsonApplication
```

## Result

Spring Boot project created successfully.
