# Financial Forecasting using Recursion (Java)

## Overview

This project demonstrates how recursion can be used to predict future financial values based on past data and annual growth rates.

---

## Files

- FinancialForecast.java
- Main.java

---

## Sample Output

```
Current Value : ₹10000.0
Growth Rate  : 10.0%
Years        : 5
Future Value : ₹16105.10
```

---

## Time Complexity

O(n)

---

## Space Complexity

O(n)

---

## Algorithm

1. Take current investment amount.
2. Take annual growth rate.
3. Recursively calculate future value.
4. Display the predicted value.
